## Convention

Each `M*.md` file includes:

1. **Goal** — what this milestone delivers
2. **Prerequisites** — hardware, prior milestones, tools
3. **Implementation steps** — ordered tasks with file paths
4. **Verification** — how to confirm it works (laptop and/or NUC)
5. **Suggested commits** — one conceptual change per commit
6. **Handoff** — what unlocks for the next milestone

See also [`development-behaviour.md`](../development-behaviour.md) for session workflow and git discipline.
