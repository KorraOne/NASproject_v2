# FrogsWork File Storage — Quick start

Thank you for choosing FrogsWork by KorraOne.

## What is in the box

- FrogsWork appliance (pre-installed)
- Power cable
- Ethernet cable (recommended)
- WiFi setup USB (if you cannot use ethernet)
- Setup code card (inside the box)

## Setup (about 10 minutes)

### 1. Connect

1. Plug in power.
2. **Recommended:** connect the ethernet cable to your router.
3. **Alternative:** use the WiFi setup USB (see card in box or `README.txt` on the USB).

### 2. Open the dashboard

On a computer on the same network, open a web browser and go to:

**http://frogswork.local**

If that does not work, check your router for the device IP address.

### 3. Claim your box

1. Enter the **setup code** from the card inside the box.
2. Create your **owner email** and **password**.
3. Choose your **timezone**.
4. You will land on the dashboard automatically.

### 4. Add your team (owner)

1. Go to **Users** and add each person who needs file access.
2. Go to **Folders** or **Permissions** to set who can access each shared folder.
3. Open **User guide** in the dashboard and copy the **employee help link** for your staff.

### 5. Staff connect on their own Windows PCs

Each employee does this on **their own computer** — you do not need to install the helper for them.

1. Open the help link your manager sent (e.g. **http://frogswork.local/help**).
2. Download **FrogsWork Helper** from that page.
3. Windows may show a SmartScreen warning — choose **More info** → **Run anyway** (the app is not code-signed in v1).
4. Sign in with the **username and password** your manager created for you.
5. Your files appear on drive **W:** in File Explorer.

## Daily use

- Leave the box powered on.
- Backups run automatically every night at 2:00 AM (your timezone).
- To restore deleted files: dashboard → **Backups**.

## Known limitations (v1)

- **Windows PCs only** for the helper app
- **Local network only** — no remote access over the internet
- **HTTP only** on your local network (not HTTPS)
- **Unsigned helper** — SmartScreen warning is expected
