# M8 — Helper app

**Status:** Not started  
**Validates:** File user installs helper on Windows PC → drives map → read/write in Explorer; reconnect after reboot.

## Goal

C# .NET 8 WPF tray application: mDNS discovery, login, drive mapping, letter conflict handling.

## Prerequisites

- [M3](M3.md)–[M4](M4.md) file users and permissions on NUC
- [M7](M7.md) helper download link on system page (or static URL)
- Windows client PC on same LAN as NUC

## Implementation steps

### 1. Discovery (`Discovery/`)

- Browse mDNS for `_frogswork._tcp`
- Fallback: manual host entry `frogswork.local` if discovery fails
- Show list if multiple appliances found (unlikely v1)

### 2. Authentication

- `POST` to appliance for file user credentials — or SMB auth only via `WNetAddConnection2`
- Store credentials securely (Windows Credential Manager)

### 3. Drive mapping (`DriveMapper/`)

- Fetch user's folder list from API endpoint (new `GET /api/helper/mounts` for authenticated file user) **or** derive from known share naming
- Default letters:
  - `U:` → private home
  - `S:`, `T:`, … → shared folders in stable order
- If letter taken: prompt user to pick alternate
- `WNetAddConnection2` with persistent flag for reboot reconnect

### 4. Tray UI

- System tray icon
- Status: connected / disconnected
- Menu: Connect, Disconnect, Change drive letters, About, Quit
- Reconnect on startup and after network restore

### 5. Packaging

- Build MSI or portable exe
- Copy artifact to `/var/lib/frogswork/helper/` on NUC
- Dashboard download serves unsigned installer (document SmartScreen warning)

### 6. API addition (if needed)

- `GET /api/helper/mounts` — returns share paths and suggested drive letters for logged-in file user
- Auth: file user credentials (basic or token exchange)

## Verification

1. Install helper on Windows client
2. App discovers `frogswork.local`
3. Login as file user → `U:` and shared drives appear
4. Create/edit file in Explorer
5. Reboot Windows → drives reconnect
6. User without permission cannot see restricted shares
